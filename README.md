# Snowflake FSI Demos

A collection of financial services industry (FSI) demos built on Snowflake.

| Demo | Description | 
|---|---|
| [regtech_demo](regtech_demo/) | AI-powered regulatory technology platform using Cortex AI (Search, Agents, ML Anomaly Detection, COMPLETE, [Cortex Code SDK](https://docs.snowflake.com/en/user-guide/cortex-code-agent-sdk/cortex-code-agent-sdk)) for balance sheet analytics, document intelligence, and pipeline compliance auditing |
| [isda_document_intelligence](isda_document_intelligence/) | ISDA Master Agreement analysis using PARSE_DOCUMENT, AI_EXTRACT, Cortex Search, and Cortex Agents for structured extraction, knowledge graphs, amendment resolution, and party-level querying via a Next.js portal |
| [sv_automation](sv_automation/) | Automated Semantic View lifecycle management using [Cortex Code Agent SDK](https://docs.snowflake.com/en/user-guide/cortex-code-agent-sdk/cortex-code-agent-sdk), query history mining, table clustering, SV generation via FAST_GENERATION, Cortex Analyst evaluation, and Cortex Search integration |
